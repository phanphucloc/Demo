ViewContainerRef

- ViewContainerRef is a container for view: EmbeddedView and Host View We can insert an EmbeddedView using the .insert() function and a host view with the createComponent() function.

- We can call variables to receive values ​​or we can change their values ​​and are similar to the functions for host view (via instance properties you can see in the code)

- Note: the components will be inserted at the bottom and not lost, if you want to delete we use the clear() function.
